# Pre-Requisites & Links

## Hardware Requirements
- Processor: Intel i3/i5 or higher (Apple Silicon M1+ also supported)
- RAM: Minimum 4GB (8GB recommended, especially if using the local fallback model)
- Storage: 10GB free disk space
- Internet connection (required for Gemini API calls and for downloading
  the local fallback model on first use)

## Software Requirements
- Operating System: Windows / Linux / macOS
- Python 3.10+
- FastAPI (installed via `requirements.txt`)
- A database is **not required** for the current feature set (the app is
  stateless — see `docs/ER_DIAGRAM.md` for the schema to add if you want
  saved history)
- Google Gemini API key
- Git & GitHub
- Visual Studio Code / PyCharm (or any editor)
